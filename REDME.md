# 📊 Excel Charts Tutorial

A beginner-friendly Excel project that demonstrates how to create different types of charts using Microsoft Excel.

## 📌 About

This project is designed for anyone who wants to learn how to create charts in Excel for data visualization. It explains the basic process of selecting data and generating charts in a simple way.

## 📂 Topics Covered

- 📈 Line Chart
- 📊 Column Chart
- 📉 Bar Chart
- 🥧 Pie Chart
- 📍 Selecting Data
- 🎨 Formatting Charts
- 📝 Adding Chart Titles
- 📅 Using Date-Based Data

## 🛠️ Software Used

- Microsoft Excel 2021

## 🎯 Features

- Easy to understand
- Beginner friendly
- Step-by-step chart creation
- Clean Excel workbook
- Practice-ready data

## 📁 Project Structure

```
Excel-Charts-Tutorial/
│
├── Excel Charts.xlsx
├── README.md
└── images/
    └── screenshot.png
```

## 📷 Screenshot

> Add your Excel chart screenshot inside the **images** folder and name it **screenshot.png**.

```md
![Project Screenshot](images/screenshot.png)
```

## 🚀 How to Use

1. Download or clone the repository.
2. Open the Excel workbook.
3. Follow the examples.
4. Practice creating charts using your own data.
